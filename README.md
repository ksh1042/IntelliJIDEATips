# IntelliJ IDEA Tips
- 인텔리제이 팁 저장소

## 1. 라이브템플릿
- 템플릿을 특정 예약어로 지정해두면 입력 시 자동완성시켜주는 기능

>![img2.png](images/img2.gif)

- ```Settings > Editor > Live Templates``` 에서 그룹, 템플릿 생성과 수정 삭제등이 가능하다.

>![img.png](images/img1.png)
